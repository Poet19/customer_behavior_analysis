# 📓 Data Analysis Scripts & Notebooks

A collection of standalone Python scripts, SQL queries, and Jupyter notebooks built while practicing and working on real-world data problems.

---

## 📂 Contents

| File/Folder | Description | Tools |
|---|---|---|
| `eda_template.ipynb` | Reusable EDA starter template | Python, Pandas |
| `sql_practice/` | SQL query collection — joins, window functions, CTEs | PostgreSQL |
| `viz_snippets/` | Reusable Matplotlib & Seaborn chart templates | Python |
| `data_cleaning/` | Common data cleaning patterns (nulls, outliers, types) | Pandas |

---

## 🧰 Tools Used

- **Python:** Pandas, NumPy, Matplotlib, Seaborn
- **SQL:** PostgreSQL — window functions, CTEs, subqueries
- **Notebooks:** Jupyter

---

## 💡 Purpose

These scripts are my working toolkit — things I've written while solving real analytical problems. Feel free to use them as references or templates.

---

## ▶️ How to Use

```bash
git clone https://github.com/Poet19/data-scripts.git
pip install pandas numpy matplotlib seaborn jupyter
jupyter notebook
```
